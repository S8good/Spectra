# 数据库浏览器优化记录

## 2025-11-10
- 引入 QtConcurrent + QFutureWatcher，实验列表查询与详情标签页统一改为后台线程加载，避免大数据量下阻塞 UI。
- 异步刷新期间禁用筛选/详情区域，并在完成后自动恢复，确保操作反馈清晰。
- 批次概览标签新增状态下拉与孔位关键字过滤，支持一键筛选/清空并快速定位大批量板位。
- 批次概览展示 SAM 光谱角与 QA 标记，并提供“仅显示需复检”开关，便于 QA 筛查。
- 批量设置对话框接入参考模板管理器，可导入/预览/校验标准光谱并应用到具体孔位；采集对话框新增 QA 面板，实时记录 SAM 告警并提供曲线对比。
