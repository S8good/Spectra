# 数据库浏览器演示数据准备指南

为方便演示新版数据库浏览器的层级导航与详情视图，可以使用随附脚本快速生成一个体量适中的 SQLite Demo 库。

## 1. 生成 Demo 库

```bash
python scripts/generate_demo_database.py --output data/demo_database.db
```

- 默认输出路径为 `data/demo_database.db`。如需覆盖已有文件，可追加 `--force` 参数。
- 脚本会写入示例项目、实验、批次、孔位及光谱集合，同时生成仪器 / 处理快照，便于在 UI 中查看快照指纹。

## 2. 在数据库浏览器中加载

1. 启动应用并打开“数据库浏览器”。
2. 在父窗口或对话框中切换数据库连接（确保 `DatabaseManager` 指向 `data/demo_database.db`）。
3. 刷新后即可看到示例项目与实验：
   - 左侧过滤面板仍可按项目名称、日期范围等筛选。
   - 右侧结果表格选中实验时，下方标签页会展示（支持角色 / 状态筛选）：
     - **实验详情**：项目 / 状态 / 负责人、创建 / 更新时间以及备注信息，方便对单个实验做快速梳理。
     - **光谱集合**：以表格列出捕获标签、角色、快照与仪器引用等核心字段，默认展示最近的光谱集合。
     - **批次概览**：批次运行名称、孔位标签、采集次数与最近采集时间，便于快速定位批量流程状态。

## 3. 演示建议

- 先展示过滤功能（按项目、类型筛选） → 加载实验详情 → 切换至光谱集合和批次概览标签。
- 可结合导出按钮或“加载光谱到分析”操作，展示结构化数据与后续分析流程的衔接。
- 演示时可以通过切换列表中的不同实验，让观众观察下方标签页的实时刷新效果（例如查看某个批量孔位的采集次数或实验备注）。
- 若需重新初始化数据，删除旧库或使用 `--force` 参数重新生成即可。
